Alterations to code and locations in the library for the implementation of the oblique scanning process:

-Acquisition logic (implements the x-shift for oblique scans)
File: python_scripts/custom_scripts/mesoSPIMcontrol_benchtop_obliquescan/mesoSPIM/src/mesoSPIM_Core.py
Link: https://github.com/Glasgow-ICG/mesoSPIM-/blob/main/python_scripts/custom_scripts/mesoSPIMcontrol_benchtop_obliquescan/mesoSPIM/src/mesoSPIM_Core.py
  What to look for: inside the image-acquisition loop the code checks state['scanning_oblique_m45'], state['scanning_oblique_p45'], and state['scanning_normal']. 
  For the oblique modes it reads z-step values (z_stepping, z_starting, z_ending) and sets move_dict['x_rel'] to +/- abs(z_stepping) depending on scan direction; 
  normal scan clears x_rel.
